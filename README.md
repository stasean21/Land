# Land

Лендинг-портфолио для услуг по визуалу маркетплейсов (Ozon, Wildberries, Яндекс Маркет) на React + Vite.

## Последние обновления (10 Апреля 2026)
**Ветка `portfolio-block-cards`**

Сегодня было реализовано:
- **Разработана секция "Работы" (`Works`)**: Вертикальный макет с разделом на категории. Полностью готов первый блок — "Дизайн и инфографика".
- **Сетка карточек 3:4**: Создана идеальная ассиметричная модульная сетка, где все загружаемые элементы сохраняют пропорции `3:4` без жесткого обрезания, выровнены "пиксель-в-пиксель".
- **Интеграция 2D Infinite Gallery**: Внедрена сложная интерактивная `InfiniteGallery` (бесконечный 2D-холст с перетаскиванием, зумом и внутренним лайтбоксом). Открывается на полный экран по кнопке «Смотреть все работы».
- **Placeholder-контент**: Сгенерированы и добавлены высококачественные AI-картинки для портфолио по разным категориям.
- **Оптимизация адаптивности**: Улучшены и вычищены CSS-стили на мобильных устройствах (Media Queries) для секций `Header`, `Hero`, `Services` и `About` (например, починено наложение полупрозрачных панелей).

## Хранение файлов в Yandex Object Storage

1. Создайте бакет в Yandex Cloud и загрузите ассеты с той же структурой путей:
`/works/...`, `/icons/...`, `/video/...`, `/photo.jpg`.
2. Сделайте доступ к файлам публичным (или через CDN с публичным read).
3. Создайте `.env.local` в корне проекта:

```bash
VITE_ASSET_BASE_URL=https://storage.yandexcloud.net/your-bucket-name
```

4. Перезапустите dev-сервер (`npm run dev`).

Если `VITE_ASSET_BASE_URL` не задан, проект продолжит использовать локальные файлы из `public/`.
